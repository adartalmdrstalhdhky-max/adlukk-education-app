# Code of Conduct

## 1. الاحترام Respect
يجب معاملة جميع المساهمين باحترام، بغض النظر عن الخلفية أو مستوى الخبرة أو الرأي.

## 2. مجتمع آمن Safe Community
لا يُسمح بأي تحرش، تمييز، أو سلوك مسيء.

## 3. الاحترافية Professionalism
تواصل بوضوح وركز على تحسين المشروع.

## 4. التعاون Collaboration
استمع للآخرين، ساعدهم، وتقبل النقد البناء.

## 5. الإبلاغ عن المشاكل Reporting Issues
إذا واجهت أي سلوك مخالف أو شاهدته، افتح Issue على GitHub أو تواصل مع المشرفين.

## 6. التنفيذ Enforcement
للمشرفين الحق في اتخاذ الإجراءات المناسبة، بما في ذلك التحذير أو إزالة المساهم من المشروع.

شكرًا لمساعدتك في الحفاظ على بيئة إيجابية.
